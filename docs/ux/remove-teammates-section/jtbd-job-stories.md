# Jobs-to-be-Done: Remove Teammates Section

## Primary Job Story

**When** I open the board to monitor multi-agent delivery,
**I want** to see only actionable information about step progress,
**So I can** quickly assess project status without filtering out noise.

## Job Dimensions

### Functional
- Scan the board to understand current progress
- Identify blocked or stuck steps
- Track completion toward milestones

### Emotional
- Feel focused, not overwhelmed
- Feel confident I'm seeing what matters
- Feel in control of project visibility

### Social
- Appear informed when stakeholders ask about progress
- Demonstrate efficient monitoring practices

## Secondary Job Stories

### JS-01: Full Board Visibility
**When** I have limited screen real estate,
**I want** the Kanban board to use maximum available width,
**So I can** see more cards without horizontal scrolling.

### JS-02: Reduced Cognitive Load
**When** scanning many cards across phases,
**I want** each card to show only essential information,
**So I can** process status quickly without parsing irrelevant details.

### JS-03: Focus on Work, Not Workers
**When** monitoring AI agent delivery,
**I want** to focus on what's being done (steps) rather than who's doing it (agents),
**So I can** assess progress based on outcomes, not implementation details.
