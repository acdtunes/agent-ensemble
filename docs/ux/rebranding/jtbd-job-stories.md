# JTBD Job Stories — Rebranding to Agent Ensemble

## Primary Job Story

> **When** I introduce my tool to other developers or share it publicly,
> **I want** the name to instantly communicate that it orchestrates AI agent teams for parallel software delivery,
> **so I can** build a distinct, memorable identity independent of the nWave framework.

## Job Dimensions

| Dimension | Description |
|-----------|-------------|
| **Functional** | Communicate purpose through the name — AI agent team orchestration |
| **Emotional** | Feel proud presenting a tool with a sharp, active identity that resonates |
| **Social** | Be recognized as a purpose-built AI agent coordination tool, not a generic "teams" utility |

## Chosen Name

**Agent Ensemble** — a coordinated group of AI agents performing together, each contributing their specialization to deliver something greater than the sum of its parts.

### Why It Fits

- **Active & dynamic**: "Ensemble" evokes coordinated performance in motion
- **AI-specific**: "Agent" clearly signals this is for AI agent teams
- **Standalone identity**: No coupling to nWave or any parent framework
- **Metaphor depth**: Musical ensembles coordinate specialized performers — maps directly to specialized agent roles (crafter, reviewer, researcher)
- **Compound clarity**: Two words, each carrying meaning — "Agent" (who) + "Ensemble" (how they work together)
